# Kling Omni 镜头图片提示词规范

Kling Omni 视频生成有三个输入通道：element_list（角色主体）、image_list（参考图片）、prompt（动作描述）。
图片 prompt 的职责是为 image_list 生成参考图——重点是**场景和构图**，而不是角色外貌。

## 核心原则：不描述角色外貌

Kling Omni 通过 element 主体锁定角色外貌和音色，image_list 的图片主要提供：
- 场景环境的视觉参考（光线、色调、空间布局）
- 画面构图参考（角色位置、前景后景关系）
- 动作起始状态参考（角色站/坐/躺的位置）

因此图片 prompt：
- **不需要**描述角色的发型、服装、面部特征（element 已锁定）
- **需要**描述场景环境（室内/室外、光线、陈设、天气）
- **需要**描述角色在画面中的位置和姿态
- **需要**描述镜头的景别和构图

## Prompt 结构

```
{场景环境描述}, {角色位置和姿态(不描述外貌)}, {景别和构图}, {光线色调}, {风格质量词}
```

## 宫格图中的每格内容

宫格图中每格对应一个镜头的参考画面：
- 保持场景环境在同一组内一致
- 每格只描述角色在该镜头的位置和动作姿态
- 不同格之间体现叙事推进（动作变化）

## 示例

### 单镜头图片（对话镜头）

角色有 element 主体，不描述外貌：
```json
{
  "storyboardId": 100,
  "engineeredPrompt": "昏暗破旧的瓦房卧室，墙角堆着几袋粮食，油灯微弱暖黄光。一个人站在门口指向前方，另一个人站在粮食袋前防御姿态。中景，室内昏暗光线，冷暖对比。photorealistic, ultra detailed, 8K, cinematic lighting",
  "negativePrompt": "模糊, 面部变形, 低质量, 水印, 多余肢体",
  "promptParams": {}
}
```

### 单镜头图片（空镜）

无角色，纯场景：
```json
{
  "storyboardId": 101,
  "engineeredPrompt": "零下八十度冰雪荒原，漫天暴风雪，远处几栋被冰封的房屋透出微弱灯光。天空阴沉，大地被厚厚白雪覆盖。全景，冷蓝色调，末世氛围。photorealistic, ultra detailed, 8K, cinematic lighting",
  "negativePrompt": "模糊, 低质量, 水印",
  "promptParams": {}
}
```

### 宫格图内容

宫格图的每格描述重点相同——场景+位置+姿态，不描述角色外貌：
```
第1格：昏暗卧室全景，墙角粮食袋，油灯微光。
第2格：一个人从门口冲入，手指前方，另一人退后一步。中景。
第3格：一个人瞪大眼睛后退，双手护在身后。特写。
第4格：一个人双手张开挡在粮食袋前。中景。
```

## 关键规则

- 有 element 主体时禁止描述角色外貌（发型/服装/面部）
- 用"一个人"、"一个身影"代替角色名和外貌描述
- 重点描述场景环境的视觉细节（光线、陈设、天气）
- 角色只描述位置和姿态（站、坐、指、退）
- 中文提示词

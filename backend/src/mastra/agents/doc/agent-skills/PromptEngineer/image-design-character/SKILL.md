# 角色多角度参考网格图 Prompt 工程规范

为单个角色生成一张 1x3 横排网格图，三格分别展示正面、侧面45°、背面，生成后由后端裁剪为三张独立参考图。

用途：Kling 主体创建（advanced-custom-elements）的 element_image_list 多角度输入。

## 布局规则（固定 1x3）

固定 1 行 3 列，无需根据数量动态调整：

| 格位 | 位置 | 视角 | 说明 |
|------|------|------|------|
| 第1格 | 左 | 正面 | 面部五官清晰可辨 |
| 第2格 | 中 | 侧面45° | 四分之三角度，面部轮廓 + 身体侧面线条 |
| 第3格 | 右 | 背面 | 后脑发型 + 服装背面 + 体态轮廓 |

禁止其他布局（如 3x1 竖排、2x2 等），裁剪程序按 1行3列 等宽切分。

## Prompt 结构

```
[布局指令] + [第1格：正面描述] + [第2格：侧面描述] + [第3格：背面描述] + [一致性] + [风格]
```

### 布局指令（固定前缀）

每条提示词必须以此开头：
```
一张1行3列的角色多角度参考网格图，共3个画面格，用粗黑分隔线均匀分隔，白色干净背景。
```

### 逐格内容

三格共用同一套角色描述（面部、体型、服装、配饰），只在开头加视角指令。

**第1格（左）— 正面**
```
第1格（左）：正面半身照，角色居中。[角色整体定位]，[面部细节：脸型/五官/肤色/发型发色/表情]，[体型]，[服装]，[配饰]。
```

**第2格（中）— 侧面45°**
```
第2格（中）：侧面四分之三角度半身照，角色面朝左前方。同一角色，[面部侧面轮廓：颧骨/下颌线/鼻梁侧影/耳部]，[发型侧面形态]，[服装侧面剪裁]，[配饰侧面可见部分]。
```

**第3格（右）— 背面**
```
第3格（右）：背面半身照，角色背对镜头。同一角色，[后脑发型：长度/层次/发尾]，[服装背面：领口后方/背部剪裁/纹理]，[体态轮廓]。
```

### 描述层级（与 image-design-character 一致）

每格的角色描述按以下层级组织：

1. **角色整体定位**: 族裔 + 性别 + 年龄段 + 体貌特征（如 `25岁左右的东亚年轻女性`）。族裔从角色的 ethnicity 字段获取，如果缺失则默认为"东亚"
2. **面部细节**: 脸型 → 五官 → 肤色 → 发型/发色 → 表情
3. **体型**: 身高体态（如 `身材高挑纤细`）
4. **服装**: 款式 + 颜色 + 材质 + 细节（从胸部到腰部可见范围）
5. **配饰**: 耳环、项链、眼镜、发饰等
6. **整体气质**: 一句话定调（如 `干练精英感`）

注意：第2格侧面和第3格背面需补充该视角特有的细节（侧面轮廓、背面发型），而非简单重复正面描述。

### 一致性约束

```
三格中同一角色的外貌、发型、服装、配饰、体型完全一致，仅拍摄角度不同，统一柔和均匀打光。
```

### 风格后缀

使用请求上下文中的「项目风格」字段作为风格描述。如果项目风格为"未设置"，默认使用"写实风格"。

```
{项目风格}，整洁网格布局，粗黑分隔线清晰，高度精细，焦点清晰，4K画质。
```

例如：项目风格为 `cinematic dark fantasy, warm golden tones, volumetric fog` 时，风格后缀为：
```
cinematic dark fantasy, warm golden tones, volumetric fog，整洁网格布局，粗黑分隔线清晰，高度精细，焦点清晰，4K画质。
```

## 输出格式（严格 JSON，不加 Markdown 代码块）

```json
{
  "characters": [
    {
      "id": 角色ID,
      "prompt": "一张1行3列的角色多角度参考网格图...",
      "negativePrompt": "模糊, 低质量, 面部变形, 多余肢体, 水印, 格子合并, 分隔线不清晰, 三格角色不一致",
      "promptParams": {"grid_rows": 1, "grid_cols": 3, "ratio": "16:9"}
    }
  ]
}
```

- 每个角色独立一条记录
- `grid_rows` 固定为 1，`grid_cols` 固定为 3
- `ratio` 固定为 `"16:9"`（横向宽图）
- `promptParams` 是裁剪程序的契约，必须准确
- 只输出 JSON，不要添加任何解释文字
- 所有提示词使用中文

## 检查清单

- 布局必须为 1行3列（grid_rows=1, grid_cols=3）
- ratio 为 "16:9"
- 第1格正面、第2格侧面45°、第3格背面，顺序不可变
- 三格角色外貌/服装/配饰完全一致
- 每格都有该视角的专属细节（不是简单重复）
- 侧面格：面部轮廓、耳部、发型侧面、服装侧面剪裁
- 背面格：后脑发型、服装背面、体态轮廓
- 白色干净背景
- 粗黑分隔线
- 中文提示词
- ID 与输入完全对应

# 内容风格分析规范

## 分析维度

### 1. 题材类型 (genre)
从以下常见类型中选择最贴切的:
- 末日生存 / 冰雪灾难 / 丧尸末世
- 甜宠爱情 / 虐恋情深 / 豪门总裁
- 悬疑推理 / 犯罪刑侦 / 密室逃脱
- 都市职场 / 商战风云 / 医疗行业
- 古风仙侠 / 宫斗权谋 / 武侠江湖
- 科幻未来 / 机甲战争 / 星际探索
- 恐怖惊悚 / 灵异鬼怪 / 心理恐惧
- 青春校园 / 成长蜕变 / 热血竞技
- 历史战争 / 民国乱世 / 抗战谍战
- 家庭伦理 / 乡村生活 / 养老育儿

可以组合多个类型，用"+"连接，如: 末日生存+家庭伦理

### 2. 情感基调 (emotionalTone)
描述故事的整体情绪:
- 压抑绝望 / 沉重悲壮 / 紧张窒息
- 温暖治愈 / 轻松幽默 / 甜蜜浪漫
- 紧张刺激 / 热血沸腾 / 惊心动魄
- 忧伤文艺 / 怀旧感伤 / 平静淡然
- 可以描述情感变化趋势，如: 前期压抑后期爆发

### 3. 视觉氛围 (visualMood)
最重要的维度，直接影响图片/视频生成的色调和光影:
- 冷色调 / 暖色调 / 高对比度 / 低饱和度
- 电影质感 / 纪实风格 / 梦幻柔焦 / 硬朗写实
- 暗角光影 / 逆光剪影 / 霓虹闪烁 / 自然光
- 阴沉灰暗 / 明亮清透 / 浓郁艳丽 / 素雅淡泊

### 4. 环境特征 (environmentKeywords)
故事发生的主要场景环境关键词:
- 冰雪覆盖 / 荒凉村庄 / 破旧房屋 / 暴风雪
- 繁华都市 / 高楼大厦 / 灯火辉煌 / 车水马龙
- 古镇小巷 / 山水田园 / 竹林溪流 / 宫殿庙宇
- 校园操场 / 教室走廊 / 宿舍阳台 / 图书馆
- 废墟荒野 / 核辐射区 / 地下掩体 / 残垣断壁

### 5. 节奏感 (pacing)
叙事节奏特征:
- 快节奏动作 / 慢节奏文艺 / 张弛有度
- 层层递进 / 反转不断 / 平铺直叙 / 倒叙插叙

## summary 字段规范

summary 是最终传给 PromptEngineer 的全局风格描述，直接影响所有图片和视频的生成:

- 控制在 80 字以内
- 聚焦**视觉风格**描述，不描述剧情
- 包含: 题材定位 + 色调 + 光影 + 环境氛围 + 整体质感
- 示例:
  - "末日冰雪灾难，极端寒冷环境，冷色调低饱和度，电影级光影，荒凉压抑，雪地废墟"
  - "现代都市甜宠，暖色调柔光，明亮清透，精致室内与城市夜景，浪漫温馨氛围"
  - "民国谍战，复古胶片质感，暗黄色调，弱光昏暗，老上海租界街巷，紧张悬疑"

# PREMIUM 精品模式剧本精修

## 适用场景

当用户选择 PREMIUM 生产模式时，已完成的二创剧本会进入精修环节。精修不是重写，而是在保持原文风格和情节的基础上提升品质。

## 精修维度

### 1. 文笔质量
- 消除口语化表达，提升书面表达的流畅度
- 替换平淡动词为更具画面感的表达（如 "走" → "缓步走来"）
- 修正语法瑕疵、重复用词

### 2. 情感张力
- 在情感高潮处加重渲染（内心独白、环境烘托、细微动作描写）
- 在平淡段落中植入微妙情感暗示（眼神、语气词、停顿）
- 对白中融入潜台词，避免直白表达

### 3. 节奏感
- 检查场景转换的节奏是否松弛有度
- 长对话段落适当插入动作/环境描写打断
- 高潮前铺垫要充分，高潮后留白要适度

### 4. 视觉化表达
- 关键场景增加可被视频呈现的细节（光影、色彩、空间感）
- 动作描写要具体到可以被分镜拆解

## 精修原则

- **不改变情节走向**：只优化表达，不增删剧情
- **不改变角色设定**：保持角色性格和说话风格一致
- **不改变叙事视角**：第一人称保持第一人称，第三人称保持第三人称
- **保守改动**：宁可少改也不要过度修饰，避免文风突变

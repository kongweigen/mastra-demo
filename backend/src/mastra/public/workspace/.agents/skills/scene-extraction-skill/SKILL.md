# 场景提取规范

## 字段定义

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| name | 是 | String | 场景名称，简洁无括号（如"林家卧室"而非"林家卧室（温暖的室内）"）|
| scene_type | 是 | String | INDOOR / OUTDOOR / VEHICLE / VIRTUAL（见下方分类说明）|
| mood | 是 | String | 场景主导情绪氛围（如"温馨"、"紧张压抑"、"阴森恐怖"）|
| environment_desc | 是 | String | 环境描述（光线、温度、陈设、声音、气味、天气等感官细节）|
| time_desc | | String | 时间描述（如"夜晚"、"黄昏"、"凌晨"）|
| location_desc | | String | 地点描述（如"老家瓦房内"、"山林深处"）|

注意：不需要 visual_summary 字段，图片提示词由后续 PromptEngineer 生成。

## scene_type 分类

| 类型 | 说明 | 典型场景 |
|------|------|---------|
| INDOOR | 室内封闭空间 | 卧室、客厅、办公室、餐厅、教室、医院病房、监狱牢房 |
| OUTDOOR | 室外开放空间 | 街道、广场、山林、海边、田野、屋顶天台、小区楼下 |
| VEHICLE | 交通工具内部 | 车内、出租车、地铁车厢、飞机舱内 |
| VIRTUAL | 非现实空间 | 梦境、回忆画面、幻想世界、手机/电脑屏幕内容、天堂/地狱 |

## 提取规则

### 核心原则：宁多勿少，宁细勿粗

短剧分镜拆解需要从已提取的场景列表中选择，**遗漏的场景会导致分镜无法绑定**。因此必须全面提取，确保覆盖剧本中每一个发生动作的空间。

**数量参考**：一集 1000-2000 字的剧本通常应提取 **5-15 个场景**。如果你只提取了 2-3 个场景，大概率遗漏了过渡场景、室内子区域或户外细分场景。请重新检查剧本。

### 什么算一个场景？

凡是剧本中**发生动作、对话或叙事的空间**，都是一个场景。判断标准：
- 有角色在此处说话或做事 → 提取
- 旁白描述了一个具体环境 → 提取
- 角色从 A 移动到 B → A 和 B 都是场景
- 闪回/梦境/回忆中的空间 → 提取（scene_type = VIRTUAL）
- 手机屏幕、视频通话画面 → 如果占据独立镜头，提取（scene_type = VIRTUAL）

### 容易遗漏的场景类型（必须注意）

| 类型 | 示例 | 说明 |
|------|------|------|
| **过渡场景** | 走廊、楼梯间、电梯、门口 | 角色移动中经过的空间，短剧中常有转场镜头 |
| **交通场景** | 车内、出租车、地铁 | 角色在交通工具中的对话/独白 |
| **通讯场景** | 电话另一端、视频通话画面 | 如果描述了对方的环境 |
| **回忆/梦境** | 童年老家、前世场景、幻觉 | 闪回画面需要独立的视觉空间 |
| **公共场所** | 医院走廊、学校操场、商场 | 与具体房间（病房、教室）是不同场景 |
| **户外细分** | 小区楼下、公园长椅旁、天台 | 不要笼统写"户外"，要具体到可辨识的位置 |
| **工作场所** | 办公隔间、会议室、前台 | 同一栋楼但空间不同 |

### 命名规则

- **场景名简洁**：短名称，不加括号注释（"极寒雪地"而非"极寒雪地（梦境/前世记忆）"）
- **场景名具体**：能区分就具体（"林家客厅"而非"客厅"，除非全剧只有一个客厅）
- **场景名一致性**：后续分镜拆解时需要精确匹配场景名，名称越简洁越不容易出错

### environment_desc 要求

必须尽可能丰富，覆盖以下感官维度（有则写，无则跳过）：
- **视觉**：光线（自然光/灯光/昏暗）、色调、空间大小、陈设物品
- **听觉**：环境音（风声、车流、音乐、安静）
- **触觉/温度**：冷暖、潮湿、干燥
- **特殊元素**：标志性物品、重要道具、窗外景色

### 合并与拆分规则

- **同一地点不同时段**：如果氛围/光线有明显差异（如白天 vs 夜晚），**拆分为两个场景**并在名称中标注（如"林家客厅-白天"和"林家客厅-夜晚"）；如果差异不大则合并
- **同一地点不同状态**：如果氛围剧变（如"客厅"白天温馨 → 晚上争吵砸东西），**拆分为两个场景**
- **不同地点分开**："林家客厅"和"张家客厅"是两个不同场景
- **大空间的子区域**：如果剧本明确区分（如"医院大厅"和"医院病房"），分开提取
- **门口/楼道/走廊**：这些过渡空间如果有戏发生，必须作为独立场景提取

### 禁止事项

- **不编造**：只提取文本中出现或强烈暗示的场景
- **不笼统**：不要写"室外"、"某个地方"，要具体
- **不遗漏**：每一处发生戏的空间都必须提取

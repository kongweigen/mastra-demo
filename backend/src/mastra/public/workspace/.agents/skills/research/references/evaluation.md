# 来源评估框架

## 可信度分级

### 第一级 — 高可信度
- 同行评审的期刊和论文
- 官方政府出版物
- 知名新闻机构（有署名记者）
- 一手来源文件

### 第二级 — 中等可信度
- 知名公司的行业报告
- 专家博客文章（附引用来源）
- 维基百科（作为起点，需验证具体内容）
- 教育机构出版物

### 第三级 — 低可信度（谨慎使用）
- 匿名论坛帖子
- 未经验证的社交媒体内容
- 内容农场和 SEO 优化文章
- 有明显商业倾向的来源

## 警示信号
- 没有作者署名
- 没有日期或信息过时
- 使用情绪化语言但缺乏证据
- 循环引用（来源互相引用）
- 非凡的主张没有非凡的证据支持

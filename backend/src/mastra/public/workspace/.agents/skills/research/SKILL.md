# 研究技能

提供进行深度研究和信息综合的方法论。

## 研究流程

### 1. 问题分解
将复杂问题拆分为可搜索的子查询：
- **谁/什么** — 识别关键实体和概念
- **何时/何地** — 建立时间和空间背景
- **为什么/如何** — 理解机制和因果关系

### 2. 搜索策略
- 先广泛搜索，再用特定词语缩小范围
- 从不同角度使用多个查询
- 同时包含专业术语和通俗描述
- 搜索相反的观点

### 3. 来源评估
使用 CRAAP 测试评估来源：
- **时效性** — 信息是否足够新？
- **相关性** — 是否回答了你的问题？
- **权威性** — 作者/出版方是谁？
- **准确性** — 信息是否可验证？
- **目的性** — 这个来源为什么存在？

### 4. 综合分析
- 识别多个来源中的共同主题
- 标注分歧及其依据
- 区分事实与观点
- 承认空白和局限

## 输出格式

将研究发现按以下结构组织：
1. **摘要** — 2-3 句话的核心回答
2. **主要发现** — 带来源的要点列表
3. **深入分析** — 对发现的更深入探讨
4. **局限性** — 哪些内容无法验证
5. **参考来源** — 使用的参考资料列表

# 第一人称改写规范

## 核心原则

将第三人称剧本转为"我"视角的沉浸式叙述，让观众代入主角。

关键：保留角色对话，不要把所有内容变成旁白。对话是短剧的核心张力。

## 改写策略

### 叙述转换

第三人称 → 第一人称：
- "林晚从床上坐起" → "我猛地从床上坐起"
- "她看到二婶推门进来" → "门突然被推开，二婶冲了进来"
- "林晚感到害怕" → "一股寒意从脚底窜上头顶"

### 保留角色对话

角色对话必须保留原文形式，通过"我"的感受过渡：

正确：
```
我还没来得及反应，二婶已经一脚踹开了门。
"把粮食交出来！"她的声音尖锐得像刀子。
我本能地挡在粮食袋前面，声音都在发抖："这是我们家的，你凭什么抢？"
```

错误（全变旁白）：
```
我听到二婶说要把粮食交出来。我告诉她这是我们家的。
→ 对话全变成了旁白复述，失去了冲突张力
```

### 内心独白插入

在关键动作/对话之间插入内心反应，但不要过多：

```
"把粮食交出来！"二婶的声音像一把钝刀。
我的手不自觉地攥紧了袋口。这些粮食，是我们一家三口最后的活路。
"这是我们家的。"我听见自己的声音在发抖，但我没有让开。
```

每个对话段落最多插入 1-2 句内心独白，不要每句都加。

### 感官细节

在场景建立和情绪转折处增加感官描写：

- 视觉："昏暗的灯光把每个人的影子拉得老长"
- 触觉："膝盖传来的寒意让我浑身颤抖"
- 听觉："门外的风声像野兽在嚎叫"
- 嗅觉："空气里弥漫着煤灰和恐惧的味道"

不要每句都加感官描写，只在以下时机使用：
- 进入新场景时（建立环境）
- 情绪转折时（强化感受）
- 高潮前的铺垫

### 旁白叙述

旁白用于：
- 时间推进："不知过了多久"
- 场景过渡："外面的风越来越大"
- 心理总结："那一刻我才明白，人心比寒冬更冷"

旁白不用于：
- 复述角色对话（保留原文对话）
- 描述角色动作（用"我看到""我听到"转换视角）

## 字数控制

目标：原文 1.2-1.5 倍（不是 2-3 倍）

扩展优先级：
1. 对话间的内心反应（最重要）
2. 场景建立的感官细节
3. 情绪转折的心理描写
4. 时间/场景过渡

不要为了凑字数而无意义扩展。每一句新增内容都应该服务于情感或画面。

## 叙述者身份

"我"= 主角（通常是女主/男主）。改写前先确认：
- 谁是主角（"我"是谁）
- 主角的性格特点（影响内心独白的语气）
- 主角与其他角色的关系

## 自检

- 角色对话是否保留了原文形式（不是旁白复述）
- 内心独白是否过多（每段对话最多 1-2 句）
- 感官描写是否过多（只在关键时刻使用）
- "我"的视角是否一致（不要突然变成全知视角）
- 字数是否在 1.2-1.5 倍范围内

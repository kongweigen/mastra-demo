# 导演讲戏本

## 剧情点 P01：末世惨死的噩梦
**时长**：15秒  
**节拍分配**：蒙太奇序列，6个子画面，每个2-3秒  

**导演阐述**：
开头是极端特写——林晚的眼球在眼皮下剧烈颤动，睫毛挂满冰晶，背景是呼啸的风雪声。镜头快速拉远，展现一家三口蜷缩在零下80度的雪地中，身上只剩破布，皮肤呈现青紫色。二婶刘翠芬的脸突然从画面左侧闯入，特写她狰狞的表情，手指死死拽着林晚母亲身上的破棉袄，指甲缝里嵌着雪泥。镜头切换到三百斤的堂弟林大宝，他骑在林晚父亲冻僵的尸体上，嘴巴疯狂啃食着半个冻硬的馒头，嘴角沾着冰碴和面包屑。林晚的视角镜头，她想张嘴呼喊，喉咙里发出破风箱般的嘶鸣，画面逐渐模糊成白色。最后一个镜头是林晚的瞳孔特写，里面反射着漫天风雪，突然瞳孔收缩，画面黑场。

## 剧情点 P02：重生回到温暖卧室
**时长**：10秒  
**节拍分配**：3个镜头，分别为3秒、4秒、3秒  

**导演阐述**：
第一个镜头是林晚的脸部特写，她猛地睁开眼睛，额头布满冷汗，呼吸急促。镜头缓慢拉远，展现她躺在温暖的床上，粉色窗帘在微风中飘动，昏黄的吊灯洒下暖光，房间里挂着红色的春联和福字。林晚颤抖着伸出手，抓住床头的手机，特写手机屏幕上的日期：2026年2月13日，腊月二十六。她的手指在屏幕上滑动，呼吸越来越急促，眼泪开始在眼眶里打转。

## 剧情点 P03：与重生的父母相认
**时长**：12秒  
**节拍分配**：4个镜头，分别为2秒、3秒、4秒、3秒  

**导演阐述**：
卧室门被猛地推开，母亲系着围裙冲进来，脸上满是焦急，父亲拿着锅铲跟在后面。林晚猛地坐起身，眼泪瞬间决堤，她扑进母亲怀里，嚎啕大哭。父亲站在旁边，手足无措地拿着锅铲，想要擦眼泪却又不敢碰。镜头特写三人的脸，母亲的手轻轻拍着林晚的背，父亲的眼睛也红了，三人的脸上混合着失而复得的狂喜和前世惨死的恐惧。

## 剧情点 P04：二婶的催命呼喊
**时长**：8秒  
**节拍分配**：2个镜头，分别为3秒、5秒  

**导演阐述**：
门外传来尖锐刺耳的公鸭嗓：“林晚！你个懒猪，几点了还不起来干活？”镜头切换到门外的视角，二婶刘翠芬叉着腰，穿着花棉袄，脸上满是不耐烦，用力砸着门。她继续喊道：“大哥！大嫂！你们家这闺女也太娇气了吧？大宝都饿了，怎么还没做饭？那半扇猪肉呢？赶紧拿出来给大宝炖了！”镜头切回室内，林晚的身体瞬间紧绷，她抬起头，眼神里充满了决绝和恨意，死死盯着门口。

## 剧情点 P05：三人觉醒的秘密
**时长**：10秒  
**节拍分配**：3个镜头，分别为2秒、5秒、3秒  

**导演阐述**：
林晚死死拽住母亲的手腕，指甲几乎嵌进肉里，她咬牙切齿地说：“妈，如果你还记得我们是怎么死的，就别去。”母亲愣住了，父亲手里的锅铲“哐当”一声掉在地上。镜头特写三人的眼睛，母亲的眼神从疑惑变成震惊，父亲的眼神从慌乱变成决绝，林晚的眼神里充满了恨意和坚定。空气仿佛凝固，窗外的鞭炮声突然变得清晰，与室内的死寂形成强烈对比。

# 导演讲戏本

## 剧情点 P01：末世惨死的噩梦
**时长**：15s  
**元数据**：回忆杀、蒙太奇、极端情绪  

**导演阐述**：
开头是极端特写，聚焦在林晚冻得青紫开裂的嘴唇，喉咙里发出破风箱般的嘶鸣，镜头缓慢拉远，展现她躺在零下80度的雪地里，身上没有任何衣物，皮肤已经冻得发黑坏死。画面硬切到二婶刘翠芬狰狞扭曲的脸，她的手死死拽着林晚妈妈那件破棉袄，指甲缝里还嵌着冰碴，镜头从她的脸向下摇，捕捉她踹向林晚肚子的动作，林晚的身体像破布一样在雪地上滑动。紧接着叠化到三百斤的堂弟林大宝骑在林晚爸爸冻僵的尸体上，疯狂啃食那半个冻硬的馒头，镜头用低角度仰拍，突出林大宝肥胖的身体和狰狞的吃相，背景是漫天风雪和远处模糊的尸体。最后画面闪白，切换到林晚猛地从床上坐起的特写，她的眼睛圆睁，瞳孔收缩，冷汗瞬间浸透了粉色睡衣，镜头环绕她的脸部一周，捕捉她从恐惧到震惊的表情变化，房间内的暖黄色灯光和窗外的鞭炮声形成强烈对比。

## 剧情点 P02：重生后的确认
**时长**：10s  
**元数据**：现实场景、情绪爆发、家庭互动  

**导演阐述**：
镜头从林晚的脸部特写拉远，展现她坐在温暖的床上，粉色窗帘在微风中轻轻晃动，昏黄的吊灯洒下柔和的光，房间里挂满了红色的春联和福字。她颤抖着拿起床头的手机，镜头推近手机屏幕，清晰显示“2026年2月13日 腊月二十六”，她的手指因为激动而微微颤抖。卧室门被猛地推开，妈妈系着碎花围裙冲进来，脸上满是焦急，爸爸手里拿着锅铲跟在后面，镜头用中景拍摄一家三口的互动，林晚扑进妈妈怀里，嚎啕大哭，爸爸手足无措地给她擦眼泪，镜头捕捉他们脸上红润有肉的脸庞，与上一世的尸体形成鲜明对比。

## 剧情点 P03：二婶的挑衅与觉醒
**时长**：10s  
**元数据**：冲突升级、觉醒、家庭共识  

**导演阐述**：
门外传来二婶刘翠芬尖锐刺耳的公鸭嗓，镜头切换到门外的视角，拍打着木门的手，指甲涂着廉价的红色指甲油。画面切回房间内，林晚听到“二婶”两个字，浑身肌肉瞬间紧绷，她的手死死拽住妈妈的手腕，指甲几乎嵌进肉里，镜头特写她的眼睛，里面充满了滔天恨意和决绝。妈妈愣住了，手里的锅铲“哐当”一声掉在地上，镜头用全景拍摄三人对视的场景，空气仿佛凝固，窗外的鞭炮声突然变得遥远，镜头缓慢推近三人的脸部，捕捉他们眼中同样的恐惧、震惊和觉醒后的决绝。

1. **千岁府卧室**：奢华而冰冷的古代卧室，雕花木床挂着深色罗帐，床边有梳妆台和圆桌，烛台烛光昏暗，晨光从窗棂透入清冷光线，色调偏青灰，氛围压抑，有囚禁感。
2. **千岁府后花园**：雪后花园，地面覆盖厚雪，枯树枝桠挂着冰凌，中央有焚烧痕迹（灰烬与残火），雪地反着冷白日光，火光橘红，氛围肃杀残酷，凄美绝望。
3. **望月楼顶**：古代高楼屋顶平台，栏杆外是夜空，清冷月光洒落，星空璀璨，楼下远处有红色灯笼光点，寒风凛冽，氛围决绝凄美，高处不胜寒。
4. **万安寺后山**：春天寺庙后山，桃花林盛开，粉白花瓣纷飞，阳光透过枝叶形成光斑，草地青翠，石径蜿蜒，氛围浪漫美好，宿命初见之地。
5. **万安寺大殿**：寺庙大殿内部，供奉佛像，香火氤氲，光束从窗棂射入形成丁达尔效应，蒲团、供桌、签筒，氛围神秘肃穆，有命运暗示感。
6. **热闹街市**：除夕夜晚古代街市，挂满红色灯笼，光影交错，行人熙攘，摊位卖年货，爆竹碎屑，暖色调，氛围喧嚣热闹，与孤独感形成对比。
7. **千岁府祠堂/院外**：祠堂内部昏暗，烛光摇曳，牌位森然排列；室外雪景清冷，枯树石阶，氛围肃穆哀伤，忏悔之地。

风格统一要求：所有场景保持一致的写实古风绘画风格，色彩协调，光影细腻，建筑细节符合中国古代制式，氛围感强烈。

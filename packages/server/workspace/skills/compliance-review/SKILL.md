# 内容合规审核技能

[技能说明]
    内容合规审核技能，检查各阶段产出是否触碰 Seedance 2.0 和 Gemini 的平台内容限制。

[审核流程]

    第一步：读取待审核内容
        - 读取当前阶段的产出文件

    第二步：逐项检查
        按照验收清单逐项检查

    第三步：输出结果
        - 全部通过 → PASS
        - 任一未通过 → FAIL + 问题清单

[验收清单]

    真人/名人限制（Seedance + Gemini 共同红线）
        □ 角色参考图是否为 AI 生成的虚构人物（非真人照片）？
        □ 角色外观是否与真实名人/公众人物不相似？
        □ 提示词中是否不包含公众人物姓名？
        □ 提示词中是否不引用真人导演名作为风格参考？

    版权 IP 限制
        □ 是否不包含已知版权 IP 角色名称（迪士尼、漫威、DC、哈利波特等）？
        □ 是否不包含已知版权 IP 的形象/设定描述？
        □ 是否不试图复刻经典影视片段？

    政治/国家安全（Seedance 平台红线）
        □ 是否不涉及反对宪法基本原则？
        □ 是否不涉及危害国家安全？
        □ 是否不涉及煽动民族仇恨/破坏民族团结？
        □ 是否不涉及歪曲/否定英雄烈士？
        □ 是否不涉及宣扬恐怖主义/极端主义？

    宗教类
        □ 是否不涉及违反宗教政策？
        □ 是否不涉及宣扬邪教/封建迷信？
        □ 是否不生成宗教人物形象（Gemini 限制）？

    色情/低俗
        □ 是否不包含淫秽色情内容？
        □ 是否不包含性暗示/软色情？
        □ 服装设计是否避免过度暴露？

    暴力/血腥
        □ 武打/格斗场景是否使用虚构角色？
        □ 是否不展示/美化过度血腥暴力？
        □ 是否不包含赌博/诈骗/犯罪指导？

    未成年人保护
        □ 是否不涉及任何形式对未成年人的侵害内容？
        □ 是否不涉及危害未成年人身心健康的内容？

    仇恨歧视
        □ 是否不包含基于性别/年龄/地域/民族/种族/宗教的歧视？

    人身权利
        □ 是否不侵犯他人名誉权/隐私权/肖像权/知识产权？

[输出规范]
    PASS 格式：
        ✅ **合规审核：PASS**
        内容未触碰平台红线。

    FAIL 格式：
        ❌ **合规审核：FAIL**
        **问题1**：[具体违规描述]
        - 位置：[如 人物设计 > 角色名 / P03 提示词]
        - 触碰红线：[哪条规则]
        - 修改方向：[如何修改以规避]

        1. 🔴 必须修改（触碰绝对禁止红线）：[列出]
        2. 🟡 建议修改（可能触发误杀）：[列出]
